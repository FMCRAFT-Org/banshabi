# BanShabi - Minecraft Spigot 1.21.4 处罚管理插件

一个功能强大的 Minecraft Spigot 1.21.4 服务器处罚管理插件，完全替代原版 Minecraft 的处罚功能。

## 功能特性

- **封禁系统**：支持临时和永久封禁，带有独立的封禁 ID
- **踢出系统**：踢出玩家并显示原因
- **禁言系统**：支持临时和永久禁言，带有独立的禁言 ID
- **警告系统**：给玩家发送警告，带有独立的警告 ID
- **查询系统**：查看封禁列表、禁言列表和警告列表
- **完全可配置**：所有消息都在 `message.yml` 中可配置
- **数据持久化**：所有处罚记录自动保存到 `punishments.yml`

## 命令说明

### /ban <玩家名> <时间> <理由> [-s]
封禁指定玩家。
- 时间格式：`1d`（1天）、`2h`（2小时）、`30m`（30分钟）、`60s`（60秒）、`-1`（永久）
- 添加 `-s` 参数可静默执行，不在公屏公示
- 示例：`/ban Steve 1d 违反服务器规则`
- 永久封禁示例：`/ban Steve -1 严重违规`
- 静默封禁示例：`/ban -s Steve 1d 违规`

### /unban <玩家名>
解封指定玩家。
- 示例：`/unban Steve`

### /kick <玩家名> <理由> [-s]
踢出指定玩家。
- 添加 `-s` 参数可静默执行，不在公屏公示
- 示例：`/kick Steve 扰乱服务器秩序`
- 静默踢出示例：`/kick -s Steve 扰乱秩序`

### /mute <玩家名> <时间> <理由> [-s]
禁言指定玩家。
- 时间格式同 /ban 命令
- 添加 `-s` 参数可静默执行，不在公屏公示
- 示例：`/mute Steve 30m 发送不当言论`
- 永久禁言示例：`/mute Steve -1 严重违规`
- 静默禁言示例：`/mute -s Steve 30m 不当言论`

### /unmute <玩家名>
解除指定玩家的禁言。
- 示例：`/unmute Steve`

### /warn <玩家名> <理由> [-s]
给玩家发送警告。
- 添加 `-s` 参数可静默执行，不在公屏公示
- 示例：`/warn Steve 请遵守服务器规则`
- 静默警告示例：`/warn -s Steve 请遵守规则`

### /unwarn <玩家名>
移除玩家的最新警告。
- 示例：`/unwarn Steve`

### /banlist [玩家名]
查看封禁列表。
- 不指定玩家名：查看所有封禁
- 指定玩家名：查看该玩家的封禁记录
- 示例：`/banlist` 或 `/banlist Steve`

### /warnlist [玩家名]
查看警告列表。
- 不指定玩家名：查看所有警告
- 指定玩家名：查看该玩家的警告记录
- 示例：`/warnlist` 或 `/warnlist Steve`

### /mutelist [玩家名]
查看禁言列表。
- 不指定玩家名：查看所有禁言
- 指定玩家名：查看该玩家的禁言记录
- 示例：`/mutelist` 或 `/mutelist Steve`

## 权限说明

- `banshabi.ban` - 封禁玩家
- `banshabi.unban` - 解封玩家
- `banshabi.kick` - 踢出玩家
- `banshabi.mute` - 禁言玩家
- `banshabi.unmute` - 解除禁言
- `banshabi.warn` - 警告玩家
- `banshabi.unwarn` - 移除警告
- `banshabi.banlist` - 查看封禁列表
- `banshabi.warnlist` - 查看警告列表
- `banshabi.mutelist` - 查看禁言列表
- `banshabi.*` - 所有权限（默认为 OP）

## 配置文件

### message.yml
所有消息都在 `message.yml` 中配置，支持颜色代码（& 符号）。

配置项包括：
- 各命令的成功/失败消息
- 处罚通知消息
- 玩家收到的处罚消息
- 列表显示格式
- 时间显示格式

### punishments.yml
自动生成的数据文件，存储所有处罚记录。不要手动编辑此文件。

## 编译和安装

1. 确保已安装 Java 21
2. 使用 Maven 编译项目：
   ```bash
   mvn clean package
   ```
3. 将生成的 JAR 文件复制到服务器的 `plugins` 文件夹
4. 重启服务器或加载插件

## 技术特性

- 使用 Java 21 编写
- 支持 Spigot 1.21.4 API
- 异步事件处理，不影响服务器性能
- 线程安全的数据存储
- 自动清理过期处罚

## 注意事项

- 此插件完全替代原版 Minecraft 的处罚功能
- 所有处罚都有独立的 ID，方便追踪和管理
- 时间格式支持天、小时、分钟、秒，`-1` 表示永久
- 所有处罚命令支持 `-s` 参数进行静默执行（不在公屏公示）
- 被封禁的玩家无法加入服务器
- 被禁言的玩家无法发送聊天消息
